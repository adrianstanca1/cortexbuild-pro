# Hostinger Node.js Environment Variables Setup Guide
# ===================================================
# Follow this guide to configure environment variables in Hostinger hPanel

## How to Set Environment Variables in Hostinger

### Method 1: Via Hostinger hPanel (Recommended)
1. Log into Hostinger hPanel: https://hpanel.hostinger.com
2. Navigate to: **Websites** → **Node.js** → Your Application
3. Click **"Environment Variables"** section
4. Add each variable below one by one

### Method 2: Via .env file (Alternative)
Upload a `.env.hostinger` file to your application root directory.
The server will automatically load it if configured.

## Required Environment Variables

### Server Configuration
PORT=3001
NODE_ENV=production

### Database Configuration (Hostinger MySQL)
# Get these from: Websites → Databases → Your Database
USE_LOCAL_DB=false
DATABASE_TYPE=mysql
DB_HOST=127.0.0.1
DB_USER=your_database_username
DB_PASSWORD=your_database_password
DB_NAME=your_database_name
DB_PORT=3306

### Security Tokens
# Generate JWT secret: node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
JWT_SECRET=your_secure_jwt_secret_min_32_chars
FILE_SIGNING_SECRET=your_secure_file_signing_secret_min_32_chars

### Storage Configuration
STORAGE_ROOT=./storage

### Application URLs
APP_URL=https://cortexbuildpro.com
CORS_ORIGIN=https://cortexbuildpro.com,https://api.cortexbuildpro.com

### Email Configuration (SendGrid)
# Get from: https://app.sendgrid.com/settings/api_keys
SENDGRID_API_KEY=your_sendgrid_api_key
EMAIL_FROM=noreply@cortexbuildpro.com

### AI API Keys
# Get from: https://aistudio.google.com/app/apikey
GEMINI_API_KEY=your_gemini_api_key

### Web Push Notifications (Optional)
# Generate at: https://vapidkeys.com/
VAPID_PUBLIC_KEY=your_vapid_public_key
VAPID_PRIVATE_KEY=your_vapid_private_key

### Feature Flags
ENABLE_DEMO_AUTH=false
MAINTENANCE_MODE=false

## Security Notes

⚠️ **IMPORTANT**:
1. Never commit actual values to git
2. Use strong, randomly generated secrets for JWT_SECRET and FILE_SIGNING_SECRET
3. Rotate API keys regularly
4. Keep database credentials secure
5. Use separate credentials for staging/production environments

## Verification

After setting variables, restart your Node.js application in hPanel:
1. Stop Application
2. Start Application
3. Check logs for any configuration errors

Test the health endpoint:
```bash
curl https://cortexbuildpro.com/api/health
```
