# CortexBuild Pro - iOS PWA Guide

## Ce este PWA?

Progressive Web App (PWA) este o aplicație web care se comportă ca o aplicație nativă pe iOS. Se instalează direct din Safari, fără App Store.

## Avantaje

✅ **Fără App Store** - Instalare directă din browser  
✅ **Actualizări automate** - Mereu cea mai recentă versiune  
✅ **Funcționează offline** - Cache inteligent  
✅ **Notificări push** - Support pentru notificări  
✅ **Iconiță pe Home Screen** - Ca o aplicație nativă  
✅ **Fullscreen** - Fără bara de adresă Safari  

## Instalare pe iOS (iPhone/iPad)

### Pasul 1: Deschide în Safari
```
1. Deschide Safari pe iPhone sau iPad
2. Navighează la https://cortexbuildpro.com (sau URL-ul tău)
```

### Pasul 2: Adaugă la Home Screen
```
1. Apasă butonul "Share" (pătrat cu săgeată în sus)
2. Derulează în jos și apasă "Add to Home Screen"
3. Confirmă numele "CortexBuild"
4. Apasă "Add" în colțul din dreapta-sus
```

### Pasul 3: Deschide aplicația
```
1. Găsește iconița CortexBuild pe Home Screen
2. Apasă pentru a deschide
3. Aplicația se va deschide fullscreen, fără interfața Safari
```

## Funcționalități iOS

### 🔔 Notificări Push
Aplicația suportă notificări push pentru:
- Task-uri noi asignate
- Actualizări proiecte
- Alertele de siguranță
- Mesaje echipă

### 📴 Mod Offline
Aplicația funcționează parțial și fără internet:
- Dashboard (date cache-uite)
- Proiecte recente
- Task-uri personale
- Formulare (se sync la reconectare)

### 🎨 Iconițe
Aplicația include iconițe optimizate pentru:
- Home Screen (180x180)
- Spotlight search (192x192)
- Settings (512x512)

## Shortcuts (Comenzi rapide)

Apasă lung pe iconița CortexBuild pentru acces rapid la:
- **Dashboard** - Vezi dashboard-ul
- **Projects** - Gestionează proiecte
- **Tasks** - Vezi toate task-urile

## Depanare

### Aplicația nu se deschide fullscreen
1. Șterge aplicația de pe Home Screen
2. Curăță cache-ul Safari (Settings > Safari > Clear History)
3. Reinstalează aplicația

### Notificările nu funcționează
1. Verifică Settings > Notifications > CortexBuild
2. Asigură-te că notificările sunt activate
3. Verifică permisiunile în aplicație

### Datele nu se actualizează
1. Trage în jos pentru refresh (pull-to-refresh)
2. Închide și redeschide aplicația
3. Verifică conexiunea la internet

## Cerințe de sistem

- **iOS 12.2+** (pentru suport PWA complet)
- **Safari** (recomandat) sau Chrome pe iOS
- **Conexiune HTTPS** (obligatoriu pentru PWA)

## Actualizări

Aplicația se actualizează automat în fundal. Când este disponibilă o versiune nouă:
- Vei vedea o notificare "Content nou disponibil!"
- Apasă "Reîncarcă" pentru a actualiza
- Sau așteaptă ca actualizarea să se aplice automat
